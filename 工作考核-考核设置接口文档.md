### 4. 新增考核规则
#### 描述
新增考核规则
#### 请求
POST /office/check/rule
```
枚举说明：
checkModule参数：
    REPORT("汇报"),
    ATTENDANCE("考勤"),
    TASK("任务"),
    DAILY("日报"),
    VISIT("拜访"),
    CUSTOM("自定义")
    
checkRuleStatus参数:
    UNEXECUTED("未执行"),
    EXECUTING("正在执行"),
    EXECUTED("执行完成")
   
period参数：
    DAY("每天"),
    WEEK("每周"),
    MONTH("每月")
    
scoreType参数：
    ADD_SCORE("加分"),
    SUB_SCORE("扣分"),
    ADD_SALARY("加工资"),
    SUB_SALARY("减工资"),
    TARGET("目标值"),
    PERIOD("周期"),
    TEMPLATE("模板")
```
```
请求body示例如下：
{
  "checkModule": "REPORT",
  "checkRuleStatus": "UNEXECUTED",
  "companyId": "",
  "startDate": "2019-04-17",
  "endDate": "2020-04-17",
  "id": "",
  "createTime": "2019-04-17 03:55:40",
  "name": "考核规则名称",
  "ruleItemDTOList": [
    {
      "checkModule": "REPORT",
      "companyId": "",
      "createTime": "2019-04-17 03:55:40",
      "id": "",
      "itemId": "考核项id",
      "itemName": "考核项名称",
      "itemNum": 0, //考核项目标值
      "isAllowRepeat": false, //是否允许重复
      "itemSubList": [
        {
          "createTime": "2019-04-17 03:55:40",
          "id": "",
          "itemId": "", //考核项id
          "itemSubId": "", //考核项子项id
          "name": "名称",
          "period": null, //周期
          "score": 0,
          "scoreType": "ADD_SCORE" //子项类型
        }
      ]
    }
  ],
  "rulePersonDTOList": [
    {
      "employeeId": "string" //参与执行人id
    },
    {
       "employeeId": "string" //参与执行人id
     }
  ]  
}
```
#### 响应
```
200
```

### 10. 考核分数处理(撤销/恢复)
#### 描述
考核分数处理(撤销/恢复)
#### 请求
POST /office/check/score/{id}/process
```
id:考核分数明细id
参数说明:
process:操作类型
    CANCEL("撤销"),
    RECOVER("恢复")
```
```
撤销分数请求
{
  "process": "CANCEL" 
}

恢复分数请求
{
  "process": "RECOVER" 
}
```
#### 响应
```json
200
```
