### 规范说明
域名：
```
接口地址
http://192.168.0.129:8089
swagger
http://192.168.0.129:8089/swagger-ui.html
```
token 验证：
```
需在http请求中添加
token : ****token****
```
http 响应状态码：
```
需要根据不同的状态码，做处理
200: 操作成功。  
500: 系统内部错误。  
401: 未授权，token过期或者无效。 
412: 参数不正确或者前置条件不足。  
501: 程序错误，抛出异常。
```

```
Gaode {
createTime (string, optional),
employeeId (string, optional),//用户ID
id (string, optional),
sid (string, optional), //高德猎鹰服务ID
tid (string, optional), //终端ID
trid (string, optional) //轨迹ID，
page 查询页数
pagesize 每页点数
ispoints 是否返回轨迹点信息
gap 补点间距
recoup 对轨迹进行补点
correction 对轨迹进行处理
employeeIds (Array[string]),//用户ID数组
qCreateTime (string)  //查询日期
}
```


### 1. 保存轨迹ID
#### 描述
app端 保存轨迹ID 
#### 请求
GET /office/gaode/saveTrid/{trid}

#### 响应
```json
200
```

### 3 终端实时监控
#### 描述
终端实时监控 
传值用户IDs
#### 请求
POST /office/gaode/lastpoint
```
{
  "employeeIds": [ //用户ID数组
    "string"
  ],
  "qCreateTime": "2018-02-01",//查询日期
  "starttime": "string", //下面的字段看猎鹰
  "endtime": "string",
  "correction": "string",
  "recoup": "string",
  "gap": "string",
  "ispoints": "string",
  "page": "string",
  "pagesize": "string"
}
```
#### 响应
```json
{
"id1":{
    猎鹰返回的结果
  },
"id2":{
    猎鹰返回的结果
  }
}
```

  ],

  "starttime": "string", //下面的字段看猎鹰

  "recoup": "string",
  "gap": "string",
  "ispoints": "string",
  "page": "string",
  "pagesize": "string"

```

{

    猎鹰返回的结果
  },{
    猎鹰返回的结果
  },{

  }],

    猎鹰返回的结果

    猎鹰返回的结果
  },{
    猎鹰返回的结果
  }]
}
```
### 6 获取当天轨迹ID
#### 描述
APP 获取当天轨迹ID 没有则返回空
传值用户ID
#### 请求
GET /office/gaode/getTrid/{userId}

#### 响应
```json
{
 "sid": "31615", //服务ID
    "tid": "64447048",//终端ID
    "trid": "asd", //轨迹ID
    "employeeId": "402880086950ab4c016950b1a4460002",
}
```
