## 1、新增客户信息
### 请求
POST /client/add
```JSON
{
  "address": "客户地址(必填)",
  "areaId": "所属区域ID(必选)",
  "belongToWhoID": "所属员工或部门ID",
  "clientBelong": "客户所属:个人(MINE)|部门(DEPARTMENT)(必选)",
  "clientGroup": "客户分组:客户(CLIENT)|商家(MERCHANT)",
  "clientLinkmanDTOList": [
    {
      "birth": "出生日期",
      "createTime": "创建时间(自动)",
      "department": "部门",
      "duty": "职务",
      "email": "email",
      "fax": "传真",
      "gender": "性别:MALE|FEMALE",
      "id": "id(自动)",
      "major": "是否主要联系人（true|false）",
      "name": "联系人名称(必填)",
      "note": "备注",
      "qq": "QQ",
      "tel": "电话(必填)",
      "weChat": "微信"
    }
  ],
  "createTime": "创建时间(自动)",
  "detailedAddress": "详细地址",
  "id": "id(自动)",
  "intentionId": "客户意向id(必选)",
  "levelId": "客户等级id(必选)",
  "name": "客户名称(必填)",
  "note": "备注",
  "operatorId": "经手人(必选,默认为自己)",
  "photos": [
    "图片的url","图片url"
  ],
  "typeId": "客户类型id(必选)"
}
```
### 响应
``` JSON
200 OK
201 Created
401 Unauthorized
403 Forbidden
404 Not Found
```
