## 1 新增费用报销
### 请求
POST /office/reimbursement
```JSON
{
  "approverId": "审批人id,必须",
  "cashierId": "出纳id,必须",
  "checkerId": "审核人id,必须",
  "copyToIds": [
    "抄送人id,零或多个"
  ],
  "photoList": [
    "图片"
  ],
  "reimbursementItemDTOList": [
    {
      "amount": "报销明细金额",
      "category": "报销类别",
      "detail": "话详细说明"
    }
  ]
}
```
### 响应
### 响应状态
```
200 OK
201 Created
401 Unauthorized
403 Forbidden
404 Not Found
```
## 2 审批
### 请求
POST /office/reimbursement/doApprove
```JSON
{
  "id": "需要审批的报销id,必须",
 "approveStatus": "审批结果,通过审批:APPROVED;驳回审批:REJECTED,必须",
 "note":"驳回原因,驳回时必须"
}
```
### 响应
### 响应状态
```
200 OK
201 Created
401 Unauthorized
403 Forbidden
404 Not Found
```
