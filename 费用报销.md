## 2 获取待我审批/我审批的
POST /office/reimbursement/getApprove
```
{
    "approveStatus":"审批状态:未审批:UNAPPROVED;已审批:APPROVED"
}
```



1 获取待我审批的列表

  "refundStatus": "审批状态,已审批,未审批,我的id(后台获取)"
