### 1. 创建群聊
#### 描述
创建群聊。  
#### 请求
POST /message/chat/group
{
  "name": "群名称",
  "memberIds": ["成员ID","成员ID"]
}
#### 响应
```json
{
  "id": "群ID",
  "name": "群名称",
  "memberIds": ["成员ID","成员ID"]
}
```
